# 01.1 Installation

Bevor du mit Java-Entwicklung beginnen kannst, benötigst du zwei Komponenten: das **JDK** (Java Development Kit) und eine **Entwicklungsumgebung (IDE)**.

## JDK (Java Development Kit)

Das JDK enthält alles, was du zum Entwickeln von Java-Programmen brauchst:

- **Compiler** (`javac`) – übersetzt `.java` → `.class`
- **Laufzeitumgebung** (`java`) – führt den Byte Code aus
- **JShell** – interaktive Java-Konsole
- **Bibliotheken** – Standard-API von Java
- Weitere Werkzeuge (Dokumentationsgenerator, Debugger, ...)

### Installation

1. Lade das JDK von [https://adoptium.net/](https://adoptium.net/) (Temurin OpenJDK, empfohlen) oder [https://www.oracle.com/java/](https://www.oracle.com/java/technologies/downloads/) herunter
2. Wähle die neueste **LTS-Version** (Long Term Support, z. B. JDK 25)
3. Führe das Installationsprogramm aus
4. **Wichtig:** Notiere dir den Installationspfad (z. B. `C:\Program Files\Eclipse Adoptium\jdk-21.0.2.13-hotspot`)

### Überprüfung der Installation

Öffne die Kommandozeile (cmd oder PowerShell) und gib folgende Befehle ein:

```
java --version
javac --version
jshell --version
```

Wenn die Versionen angezeigt werden, ist das JDK korrekt installiert.

### Umgebungsvariable JAVA_HOME (optional, aber empfohlen)

Einige Programme (wie IntelliJ) finden das JDK automatisch. Falls nicht:

1. Systemsteuerung → Erweiterte Systemeinstellungen → Umgebungsvariablen
2. Neue Systemvariable `JAVA_HOME` mit Wert `C:\Program Files\Eclipse Adoptium\jdk-21.0.2.13-hotspot` (angepasst an deinen Pfad)
3. `%JAVA_HOME%\bin` zur `Path`-Variable hinzufügen

## IntelliJ IDEA

IntelliJ ist eine professionelle Entwicklungsumgebung (IDE) für Java, vergleichbar mit PyCharm für Python.

Als Schüler:in hast du Zugang zu einem kostenlosen **JetBrains Studenten-Package** – damit kannst du die **Ultimate Edition** von IntelliJ IDEA nutzen, die deutlich mehr Funktionen bietet als die Community Edition.

Beantrage das Studenten-Package auf [https://www.jetbrains.com/shop/eform/students](https://www.jetbrains.com/shop/eform/students) mit deiner **Schul-E-Mail-Adresse**. 
Nach Bestätigung erhältst du ein kostenloses Abonnement für alle JetBrains-Produkte (gültig für 1 Jahr, verlängerbar).

### Installation

1. Lade IntelliJ von [https://www.jetbrains.com/idea/download/](https://www.jetbrains.com/idea/download/) herunter und wähle die **Ultimate Edition** (im Download-Bereich oben)
2. Führe das Installationsprogramm aus
3. Wichtig: Hake bei der Installation "Add `bin` to PATH" und ".java" an
4. Melde dich nach der Installation in der IDE mit deinem JetBrains-Konto an – die Lizenz wird automatisch aktiviert

## Zusammenfassung

### Wichtige Begriffe
- JDK (Java Development Kit)
- JRE (Java Runtime Environment)
- IntelliJ IDEA
- OpenJDK / Adoptium Temurin
- JAVA_HOME

### Das sollst du können
- Das JDK installieren und überprüfen
- IntelliJ installieren
- Die Kommandozeilen-Tools `java`, `javac` und `jshell` verwenden
