# 02.0 Erste Schritte mit JShell

**JShell** ist die interaktive Java-Konsole, ähnlich der Python-Konsole, die du aus Thonny kennst. Sie wurde in Java 9 eingeführt und ermöglicht es, Java-Code zeilenweise zu testen, ohne ein vollständiges Programm schreiben zu müssen.

## JShell starten

Öffne die Kommandozeile und gib ein:

```
jshell
```

Du siehst den JShell-Prompt:

```
|  Welcome to JShell -- Version 21
|  For an introduction type: /help intro

jshell>
```

## Erste Berechnungen

Wie in der Python-Konsole kannst du direkt rechnen:

```java
jshell> 2 + 3
$1 ==> 5

jshell> 3.14159 * 2.5
$2 ==> 7.853975

jshell> 10 / 3
$3 ==> 3
```

**Achtung:** Anders als Python gibt Java bei `10 / 3` den Wert `3` zurück, da beide Operanden ganze Zahlen (`int`) sind. Für ein genaues Ergebnis muss mindestens eine Zahl als Gleitkommazahl geschrieben werden:

```java
jshell> 10.0 / 3
$4 ==> 3.3333333333333335
```

## Variablen in JShell

Anders als in Python musst du in Java den **Typ einer Variable angeben**:

```java
jshell> int alter = 25
alter ==> 25

jshell> double koerpergroesse = 1.85
koerpergroesse ==> 1.85

jshell> String name = "Anna"
name ==> "Anna"
```

In Python hättest du einfach `alter = 25` geschrieben. Java verlangt die Angabe des Typs (`int`, `double`, `String`, ...).

## Semikolons

In Java enden **alle Anweisungen mit einem Semikolon** (`;`). Das ist der größte syntaktische Unterschied zu Python.

```java
// Java
int x = 5;      // Semikolon am Ende
System.out.println(x);
```

```python
# Python
x = 5           # kein Semikolon
print(x)
```

## Text auf der Konsole ausgeben

In Python verwendest du `print()`, in Java heißt die Methode `System.out.println()`:

```java
jshell> System.out.println("Hallo, Welt!")
Hallo, Welt!
```

## JShell-Befehle

JShell bietet eigene Kommandos, die mit `/` beginnen:

| Befehl | Beschreibung |
|--------|-------------|
| `/help` | Hilfe anzeigen |
| `/list` | Alle bisherigen Eingaben anzeigen |
| `/vars` | Alle definierten Variablen anzeigen |
| `/methods` | Alle definierten Methoden anzeigen |
| `/exit` | JShell beenden |
| `/edit` | Editor für die letzte Eingabe öffnen |
| `/drop 1` | Die Eingabe Nr. 1 löschen |

## Methoden in JShell

Du kannst auch Methoden definieren und testen:

```java
jshell> int quadrat(int x) {
   ...>     return x * x;
   ...> }
|  created method quadrat(int)

jshell> quadrat(7)
$5 ==> 49
```

Das `return`-Keyword gibt einen Wert zurück – genau wie in Python.

## Zusammenfassung

### Wichtige Begriffe
- JShell
- Semikolon (`;`)
- `System.out.println()`

- JShell starten und verwenden
