# 05.4 Klassenmethoden, Instanzmethoden, Klassenattribute, Instanzattribute

Das `static`-Schlüsselwort in Java ist der zentrale Unterschied zwischen "gehört zur Klasse" und "gehört zum Objekt". In Python kennst du das Konzept von `@classmethod` und `@staticmethod`.

## Instanzmethoden und Instanzattribute

Instanzmethoden und -attribute gehören zu einem **konkreten Objekt**. Sie werden ohne `static` deklariert:

```java
public class Patient {
    private String name;        // Instanzattribut
    private int alter;          // Instanzattribut

    public Patient(String name, int alter) {
        this.name = name;
        this.alter = alter;
    }

    public void ausgabe() {     // Instanzmethode
        System.out.println(name + " (" + alter + ")");
    }
}
```

Verwendung:

```java
Patient p1 = new Patient("Anna", 25);
Patient p2 = new Patient("Ben", 30);
p1.ausgabe();  // Anna (25)
p2.ausgabe();  // Ben (30)
```

## Klassenmethoden und Klassenattribute (static)

Mit `static` deklarierte Member gehören zur **Klasse selbst**, nicht zu einzelnen Objekten:

```java
public class Patient {
    private static int anzahlPatienten = 0;  // Klassenattribut
    private String name;

    public Patient(String name) {
        this.name = name;
        anzahlPatienten++;  // wird bei jedem neuen Objekt erhöht
    }

    public static int getAnzahlPatienten() {  // Klassenmethode
        return anzahlPatienten;
    }
}
```

Verwendung:

```java
Patient p1 = new Patient("Anna");
Patient p2 = new Patient("Ben");

System.out.println(Patient.getAnzahlPatienten());  // 2
```

## Wann verwendet man static?

`static` verwendest du, wenn eine Methode oder ein Attribut **nicht an ein bestimmtes Objekt gebunden** ist:

| Verwendung | Beispiel |
|------------|----------|
| **Zähler/Statistiken** | Anzahl aller erzeugten Objekte einer Klasse |
| **Hilfsmethoden** | Mathematische Berechnungen ohne Objektzustand |
| **Konstanten** | Fixe Werte wie `MAX_GROESSE`, `PI` |
| **Factory-Methoden** | Methoden, die ein Objekt erstellen (`valueOf()`, `of()`) |
| **main-Methode** | Der Einstiegspunkt des Programms muss static sein |

Eine gute Faustregel: Wenn eine Methode **keine Instanzattribute** verwendet und **kein `this`** braucht, sollte sie `static` sein.

**Falscher Einsatz von static** (vermeide das):

```java
public class Patient {
    private static String name;  // FALSCH! Alle Patienten heißen gleich
}
```

`static`-Attribute gehören zur Klasse, nicht zum Objekt – alle Instanzen teilen sich denselben Wert.

## Wichtige Regeln für static

| Zugriff | Instanzmethode | Klassenmethode (static) |
|---------|:---:|:---:|
| Kann Instanzattribute lesen/schreiben | ✓ | ✗ |
| Kann Klassenattribute lesen/schreiben | ✓ | ✓ |
| Kann andere Instanzmethoden aufrufen | ✓ | ✗ |
| Kann andere Klassenmethoden aufrufen | ✓ | ✓ |
| Hat Zugriff auf `this` | ✓ | ✗ |

```java
public class Beispiel {
    private int instanzAttribut;
    private static int klassenAttribut;

    public void instanzMethode() {
        System.out.println(instanzAttribut);  // OK
        System.out.println(klassenAttribut);  // OK
        System.out.println(this.instanzAttribut);  // OK
    }

    public static void klassenMethode() {
        // System.out.println(instanzAttribut);  // Compiler-Fehler!
        System.out.println(klassenAttribut);  // OK
        // System.out.println(this.instanzAttribut);  // Compiler-Fehler!
    }
}
```

## Konstanten mit static final

In Python definierst du Konstanten auf Modulebene (z. B. `PI = 3.14159`). In Java verwendest du `static final`:

```java
public class MedizinischeKonstanten {
    public static final double NORMAL_TEMPERATUR = 37.0;
    public static final double MAX_BLUTDRUCK_SYSTOLISCH = 140;
    public static final int ID_LAENGE = 10;
}
```

Verwendung:

```java
System.out.println(MedizinischeKonstanten.NORMAL_TEMPERATUR);
```

## Vergleich Python vs. Java

| Konzept | Python | Java |
|---------|--------|------|
| Instanzmethode | `def methode(self):` | `void methode() { }` |
| Klassenmethode | `@classmethod def methode(cls):` | `static void methode() { }` |
| Klassenattribut | `attribut = wert` (in Klasse) | `static typ attribut = wert;` |
| Konstante | `KONSTANTE = wert` (Konvention) | `static final TYP KONSTANTE = wert;` |
| Zugriff auf Klassenmember | `Klasse.member` | `Klasse.member` |

## Zusammenfassung

### Wichtige Begriffe
- `static`
- Klassenmethode / Klassenattribut
- Instanzmethode / Instanzattribut
- `final` (Konstante)
- `static final`

### Das sollst du können
- Den Unterschied zwischen Instanz- und Klassenmembern erklären
- `static`-Methoden und -Attribute definieren
- Die Zugriffsregeln für `static` anwenden
- Entscheiden, wann `static` sinnvoll ist (Hilfsmethoden, Konstanten, Zähler)
- Konstanten mit `static final` definieren
