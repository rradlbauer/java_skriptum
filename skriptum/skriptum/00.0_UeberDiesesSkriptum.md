# 00.0 Über dieses Skriptum

Dieses Skriptum begleitet den Einstieg in die Programmiersprache Java.

Es setzt **fundierte Python-Kenntnisse** voraus, 
wie du sie in den ersten beiden Jahrgängen gelernt hast. 
Der Fokus liegt auf den Unterschieden und Gemeinsamkeiten zwischen Python und Java. 
Die Konzepte werden oft im Vergleich zu Python erklärt.

## Aufbau

Das Skriptum umfasst folgende Kapitel:

Jedes Kapitel enthält:

- Erklärungen mit Code-Beispielen
- Übungsaufgaben - wenn möglich mit medizinischem Kontext
- Eine Zusammenfassung der wichtigsten Begriffe

## Konventionen

- Code-Beispiele werden in Java mit Syntax-Highlighting dargestellt
- Wichtige Begriffe sind **fett** hervorgehoben
- Java-Code wird in `Schriftart für Code` dargestellt
- Dateinamen und Pfade ebenfalls in `Code-Schriftart`

## Übungen

Die Übungen befinden sich im Ordner `uebungen/`. 
Sie sind nach dem Schema `UE_XX.Y_Thema.md` benannt. 
Die Übungen sind so gestaltet, 
dass sie den Stoff des jeweiligen Kapitels vertiefen.

## Voraussetzungen

- Grundlegende Programmierkenntnisse in Python
- Verständnis von Variablen, Kontrollstrukturen, Funktionen und grundlegender OOP
