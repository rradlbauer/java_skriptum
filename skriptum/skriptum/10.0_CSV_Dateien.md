# 10.0 CSV-Dateien lesen und schreiben

**CSV** (Comma-Separated Values) ist ein einfaches Textformat zum Austausch von Tabellendaten – häufig verwendet in der Medizin (Patientendaten, Laborwerte, Abrechnungsdaten).

```
Name,Alter,Diagnose
Anna Müller,34,Hypertonie
Ben Bauer,28,Diabetes
Clara Schmidt,45,Fraktur
```

In diesem Kapitel lernst du, CSV-Dateien ohne externe Bibliotheken zu lesen und zu schreiben.

## CSV-Datei schreiben

Zum Schreiben verwendest du `FileWriter` und `PrintWriter`:

```java
import java.io.PrintWriter;
import java.io.FileWriter;
import java.io.IOException;

public class CsvSchreiben {
    public static void main(String[] args) {
        String datei = "patienten.csv";

        try (PrintWriter writer = new PrintWriter(new FileWriter(datei))) {
            // Header-Zeile
            writer.println("Name,Alter,Diagnose");

            // Datenzeilen
            writer.println("Anna Müller,34,Hypertonie");
            writer.println("Ben Bauer,28,Diabetes");
            writer.println("Clara Schmidt,45,Fraktur");

            System.out.println("CSV-Datei erstellt: " + datei);
        } catch (IOException e) {
            System.err.println("Fehler beim Schreiben: " + e.getMessage());
        }
    }
}
```

`try-with-resources` schließt die Datei automatisch – auch bei einem Fehler.

### Mehrere Patienten aus einer Liste schreiben

```java
import java.io.PrintWriter;
import java.io.FileWriter;
import java.io.IOException;
import java.util.List;

public class CsvSchreibenListe {
    public static void main(String[] args) {
        List<String[]> patienten = List.of(
            new String[]{"Anna Müller", "34", "Hypertonie"},
            new String[]{"Ben Bauer", "28", "Diabetes"},
            new String[]{"Clara Schmidt", "45", "Fraktur"}
        );

        try (PrintWriter writer = new PrintWriter(new FileWriter("patienten.csv"))) {
            writer.println("Name,Alter,Diagnose");
            for (String[] p : patienten) {
                writer.println(String.join(",", p));
            }
        } catch (IOException e) {
            System.err.println("Fehler: " + e.getMessage());
        }
    }
}
```

`String.join(",", p)` fügt die Felder mit Komma zusammen – das ersetzt eine manuelle Schleife.

## CSV-Datei lesen

Für das Lesen von CSV-Dateien verwendest du den `Scanner` – du kennst ihn bereits von der Konsolen-Eingabe. Alternativ kannst du auch `BufferedReader` einsetzen.

### Datei lesen mit Scanner

Der `Scanner` arbeitet bei Dateien genauso wie bei der Tastatureingabe: Statt `new Scanner(System.in)` übergibst du einfach `new File(datei)`.

```java
import java.util.Scanner;
import java.io.File;
import java.io.IOException;

public class CsvLesenMitScanner {
    public static void main(String[] args) {
        String datei = "patienten.csv";

        try (Scanner scanner = new Scanner(new File(datei))) {
            String header = scanner.nextLine();          // Kopfzeile überspringen
            System.out.println("Header: " + header);

            while (scanner.hasNextLine()) {
                String zeile = scanner.nextLine();
                String[] felder = zeile.split(",");
                System.out.println("Gelesen: " + String.join(" | ", felder));
            }
        } catch (IOException e) {
            System.err.println("Fehler beim Lesen: " + e.getMessage());
        }
    }
}
```

Ausgabe:
```
Header: Name,Alter,Diagnose
Gelesen: Anna Müller | 34 | Hypertonie
Gelesen: Ben Bauer | 28 | Diabetes
Gelesen: Clara Schmidt | 45 | Fraktur
```

### Variante mit BufferedReader

Falls du rein zeilenweise lesen möchtest, kannst du auch `BufferedReader` verwenden:

```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

try (BufferedReader reader = new BufferedReader(new FileReader(datei))) {
    String header = reader.readLine();          // Kopfzeile überspringen
    System.out.println("Header: " + header);

    String zeile;
    while ((zeile = reader.readLine()) != null) {
        String[] felder = zeile.split(",");
        // ...
    }
}
```

### Scanner vs. BufferedReader

| Aspekt | `Scanner` | `BufferedReader` |
|--------|-----------|-----------------|
| Einfachheit | `nextLine()` wie bei Tastatureingabe | `readLine()` |
| Zahlen parsen | direkt mit `nextInt()`, `nextDouble()` | manuell mit `Integer.parseInt()` |
| Trennzeichen | konfigurierbar mit `useDelimiter()` | keine |
| Datei-Quelle | `new File(datei)` | `new FileReader(datei)` |
| Geschwindigkeit | etwas langsamer | schneller (gepuffert) |

**Faustregel:** Verwende `Scanner` – er ist einfacher und dir bereits vertraut. Greife zu `BufferedReader`, wenn du viele große Dateien verarbeiten musst.

### Scanner mit Trennzeichen für CSV

Der Scanner kann auch direkt mit einem Trennzeichen (Delimiter) konfiguriert werden – so entfällt `split(",")`:

```java
try (Scanner scanner = new Scanner(new File(datei))) {
    scanner.useDelimiter("[,\n]");          // Trennung bei Komma oder Zeilenende
    // oder kürzer:
    scanner.useDelimiter(",");

    String header = scanner.nextLine();    // Kopfzeile überspringen

    while (scanner.hasNextLine()) {
        // Eine ganze Zeile lesen und dann parsen
        String zeile = scanner.nextLine();
    }
}
```

Achtung: `useDelimiter(",")` alleine ist für CSV ungeeignet, da Zeilenumbrüche nicht als Trennzeichen erkannt werden. Meist ist die Kombination aus `nextLine()` und `split()` einfacher.

#### Scanner mit Zahlenparsen

Der Scanner kann Zahlen direkt erkennen – nützlich wenn du gemischte Datentypen hast:

```java
try (Scanner scanner = new Scanner(new File("zahlen.csv"))) {
    while (scanner.hasNextLine()) {
        // Scanner mit zwei Trennzeichen: Komma und Zeilenumbruch
        scanner.useDelimiter(",|\n");

        String name = scanner.next();
        int alter = scanner.nextInt();       // direkt als int
        double groesse = scanner.nextDouble(); // direkt als double
        scanner.nextLine();                  // Rest der Zeile konsumieren

        System.out.printf("%s (%d Jahre, %.2f m)%n", name, alter, groesse);
    }
}
```

Das ist kürzer als `Integer.parseInt()` und `Double.parseDouble()` – aber auch fehleranfälliger, wenn die Daten nicht exakt dem erwarteten Format entsprechen.

## CSV mit Objekten

Noch praktischer: Du wandelst die CSV-Daten direkt in Objekte um:

```java
public class Patient {
    private String name;
    private int alter;
    private String diagnose;

    public Patient(String name, int alter, String diagnose) {
        this.name = name;
        this.alter = alter;
        this.diagnose = diagnose;
    }

    @Override
    public String toString() {
        return name + " (" + alter + ", " + diagnose + ")";
    }
}
```

Einlesen und in `Patient`-Objekte umwandeln:

```java
public class CsvZuPatienten {
    public static void main(String[] args) {
        List<Patient> patienten = new ArrayList<>();

        try (BufferedReader reader = new BufferedReader(new FileReader("patienten.csv"))) {
            reader.readLine();  // Header überspringen
            String zeile;

            while ((zeile = reader.readLine()) != null) {
                String[] f = zeile.split(",");
                String name = f[0];
                int alter = Integer.parseInt(f[1]);
                String diagnose = f[2];
                patienten.add(new Patient(name, alter, diagnose));
            }
        } catch (IOException e) {
            System.err.println("Fehler: " + e.getMessage());
        }

        for (Patient p : patienten) {
            System.out.println(p);
        }
    }
}
```

## CSV ausgeben mit Streams

Mit der Streaming API (Kapitel 09.3) kannst du CSV-Daten elegant verarbeiten:

```java
List<Patient> patienten = new ArrayList<>();

try (BufferedReader reader = new BufferedReader(new FileReader("patienten.csv"))) {
    patienten = reader.lines()                    // Stream<String>
        .skip(1)                                   // Header überspringen
        .map(zeile -> zeile.split(","))
        .map(f -> new Patient(f[0], Integer.parseInt(f[1]), f[2]))
        .toList();
} catch (IOException e) {
    System.err.println("Fehler: " + e.getMessage());
}
```

`reader.lines()` gibt einen Stream der Zeilen zurück – damit entfällt die manuelle `while`-Schleife.

## Wichtige Hinweise

### String.split() – Einschränkungen

`split(",")` funktioniert gut, solange die Werte **keine Kommas enthalten**. Enthält ein Feld ein Komma (z. B. "Müller, Anna"), wird es fälschlich getrennt. Richtige CSV-Parser können das mit Anführungszeichen:

```csv
Name,Alter
"Müller, Anna",34
```

Einfaches `split(",")` würde hier falsch trennen. Für dieses Kapitel reicht die Annahme, dass Felder keine Kommas enthalten.

### Zahlen parsen

Aus der CSV werden alle Werte als Strings gelesen. Für numerische Werte musst du sie umwandeln:

```java
int alter = Integer.parseInt(f[1]);
double wert = Double.parseDouble(f[2]);
```

Fehlerhafte Zahlen (`Integer.parseInt("abc")`) lösen eine `NumberFormatException` aus – fange sie mit `try/catch`.

- CSV (Comma-Separated Values)
