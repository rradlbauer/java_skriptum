# 09.4 Streaming API

Die **Streaming API** (Java 8+) ermöglicht es, Daten in einer deklarativen Art zu verarbeiten – ähnlich wie List Comprehensions oder `map()`/`filter()` in Python.

## Was ist ein Stream?

Ein **Stream** ist kein weiterer Collection-Typ, sondern ein **Datenstrom**, der eine Folge von Elementen verarbeitet. Streams verändern die ursprüngliche Datenquelle nicht und können nur einmal durchlaufen werden.

```java
import java.util.List;
import java.util.ArrayList;
import java.util.stream.Stream;

List<String> namen = List.of("Anna", "Ben", "Clara", "David");

Stream<String> stream = namen.stream();  // Stream erzeugen
stream.forEach(System.out::println);     // alle Elemente ausgeben
// stream.forEach(...);                  // Fehler! Stream bereits konsumiert
```

Vergleich mit Python:

```python
# Python
namen = ["Anna", "Ben", "Clara", "David"]
for name in namen:
    print(name)
```

## Aufbau einer Stream-Operation

Ein Stream durchläuft drei Phasen:

```
Quelle (Liste, Set, Array, ...)
    → Zwischenoperation(en) (intermediate operations)
        → Terminale Operation (liefert Ergebnis)
```

- **Quelle:** z. B. `list.stream()`, `set.stream()`, `Arrays.stream(array)`
- **Zwischenoperationen:** `filter()`, `map()`, `sorted()`, `distinct()`, ... – liefern einen neuen Stream
- **Terminale Operation:** `collect()`, `forEach()`, `count()`, `toList()`, ... – liefern das Ergebnis

## Wichtige Zwischenoperationen

### filter – Elemente auswählen

Wie `filter()` in Python oder eine Bedingung in einer List Comprehension:

```java
List<Integer> zahlen = List.of(1, 2, 3, 4, 5, 6);

List<Integer> gerade = zahlen.stream()
    .filter(n -> n % 2 == 0)
    .toList();                          // [2, 4, 6]
```

```python
# Python
zahlen = [1, 2, 3, 4, 5, 6]
gerade = [n for n in zahlen if n % 2 == 0]
```

### map – Elemente transformieren

Wie `map()` in Python:

```java
List<String> namen = List.of("anna", "ben", "clara");

List<String> gross = namen.stream()
    .map(n -> n.toUpperCase())
    .toList();                          // ["ANNA", "BEN", "CLARA"]
```

```python
# Python
namen = ["anna", "ben", "clara"]
gross = [n.upper() for n in namen]
```

### sorted – sortieren

```java
List<Integer> zahlen = List.of(5, 1, 4, 2, 3);

List<Integer> sortiert = zahlen.stream()
    .sorted()
    .toList();                          // [1, 2, 3, 4, 5]

// Mit Comparator (absteigend)
List<Integer> absteigend = zahlen.stream()
    .sorted((a, b) -> b - a)
    .toList();                          // [5, 4, 3, 2, 1]
```

### distinct – Duplikate entfernen

```java
List<Integer> mitDuplikaten = List.of(1, 2, 2, 3, 3, 3);

List<Integer> unique = mitDuplikaten.stream()
    .distinct()
    .toList();                          // [1, 2, 3]
```

### limit und skip – begrenzen

```java
List<Integer> zahlen = List.of(1, 2, 3, 4, 5);

List<Integer> ersteDrei = zahlen.stream()
    .limit(3)
    .toList();                          // [1, 2, 3]

List<Integer> ohneErsteZwei = zahlen.stream()
    .skip(2)
    .toList();                          // [3, 4, 5]
```

## Wichtige terminale Operationen

### collect – in eine Collection sammeln

```java
import java.util.stream.Collectors;

List<String> namen = List.of("Anna", "Ben", "Clara");

List<String> gefiltert = namen.stream()
    .filter(n -> n.startsWith("A"))
    .collect(Collectors.toList());      // ["Anna"]

Set<String> alsSet = namen.stream()
    .collect(Collectors.toSet());       // Set ohne Duplikate

Map<String, Integer> alsMap = namen.stream()
    .collect(Collectors.toMap(
        n -> n,                         // Schlüssel: der Name selbst
        n -> n.length()                 // Wert: Länge des Namens
    ));
```

Seit Java 16 gibt es die Kurzform `toList()`:

```java
List<String> ergebnis = stream.toList();  // kürzer als .collect(Collectors.toList())
```

### count – zählen

```java
long anzahl = namen.stream()
    .filter(n -> n.length() > 3)
    .count();
```

### forEach – für jedes Element ausführen

```java
namen.stream()
    .forEach(System.out::println);
```

### anyMatch / allMatch / noneMatch – Prüfen

```java
List<Integer> zahlen = List.of(1, 2, 3, 4, 5);

boolean hatGerade = zahlen.stream()
    .anyMatch(n -> n % 2 == 0);          // true (mindestens eine gerade Zahl)

boolean allePositiv = zahlen.stream()
    .allMatch(n -> n > 0);               // true (alle > 0)

boolean keineNegativ = zahlen.stream()
    .noneMatch(n -> n < 0);              // true (keine negativen)
```

### reduce – auf einen Wert reduzieren

```java
List<Integer> zahlen = List.of(1, 2, 3, 4, 5);

int summe = zahlen.stream()
    .reduce(0, (a, b) -> a + b);         // 15 (0 + 1 + 2 + 3 + 4 + 5)

int produkt = zahlen.stream()
    .reduce(1, (a, b) -> a * b);         // 120 (1 * 1 * 2 * 3 * 4 * 5)
```

## Kombinierte Beispiele

### Patienten nach Diagnose filtern und sortieren

```java
List<Patient> allePatienten = List.of(
    new Patient("Anna", "Grippe", 25),
    new Patient("Ben", "Diabetes", 30),
    new Patient("Clara", "Grippe", 22),
    new Patient("David", "Fraktur", 45)
);

List<String> grippeNamen = allePatienten.stream()
    .filter(p -> p.getDiagnose().equals("Grippe"))
    .map(Patient::getName)
    .sorted()
    .toList();
// Ergebnis: ["Anna", "Clara"] (alphabetisch sortiert)
```

### Gruppierung mit groupingBy

```java
import java.util.Map;
import java.util.List;
import java.util.stream.Collectors;

Map<String, List<Patient>> nachDiagnose = allePatienten.stream()
    .collect(Collectors.groupingBy(Patient::getDiagnose));

// Ergebnis:
// "Grippe"  → [Anna (25), Clara (22)]
// "Diabetes" → [Ben (30)]
// "Fraktur"  → [David (45)]
```

```python
# Python Vergleich
from itertools import groupby
from operator import attrgetter

# Ähnlich zu groupingBy
nach_diagnose = {}
for p in patienten:
    nach_diagnose.setdefault(p.diagnose, []).append(p)
```

## Methoden-Referenzen `::`

Du hast in den Beispielen `System.out::println` und `Patient::getName` gesehen. Das sind **Methoden-Referenzen** – eine kürzere Schreibweise für Lambda-Ausdrücke:

| Lambda | Methoden-Referenz |
|--------|------------------|
| `s -> System.out.println(s)` | `System.out::println` |
| `p -> p.getName()` | `Patient::getName` |
| `s -> s.toUpperCase()` | `String::toUpperCase` |
| `n -> new Patient(n)` | `Patient::new` |

## Zusammenfassung

### Wichtige Begriffe
- Stream
- Zwischenoperation (intermediate)
- Terminale Operation (terminal)
- `filter` / `map` / `sorted` / `distinct`
- `collect` / `count` / `forEach`
- `groupingBy`
- Methoden-Referenz `::`
- Lambda-Ausdruck `(x -> ...)`

### Das sollst du können
- Einen Stream aus einer Collection erzeugen
- Zwischenoperationen wie `filter`, `map`, `sorted` anwenden
- Terminale Operationen wie `collect`, `count`, `forEach` nutzen
- Daten mit `groupingBy` gruppieren
- Den Unterschied zwischen Streams und Collections erklären
- Lambda-Ausdrücke und Methoden-Referenzen verwenden
