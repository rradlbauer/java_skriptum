# 05.1 Zugriffsmodifizierer und Datenkapselung

Ein zentrales Konzept der OOP ist die **Datenkapselung** (encapsulation): Der innere Zustand eines Objekts soll vor direktem Zugriff von außen geschützt werden.

## Zugriffsmodifizierer

Java bietet vier Zugriffsmodifizierer:

| Modifizierer | Zugriff innerhalb der Klasse | Zugriff innerhalb des Packages | Zugriff in Unterklassen | Zugriff von überall |
|-------------|:---:|:---:|:---:|:---:|
| `private` | ✓ | – | – | – |
| (default/package-private) | ✓ | ✓ | – | – |
| `protected` | ✓ | ✓ | ✓ | – |
| `public` | ✓ | ✓ | ✓ | ✓ |

## Private Attribute – Der Standard für Datenkapselung

In gutem Java-Code sind Attribute fast immer `private`:

```java
public class Patient {
    private String name;
    private int alter;
    private double koerpergroesse;

    public void ausgabe() {
        System.out.println(name + ", " + alter + " Jahre");
    }
}
```

Der Zugriff von außen auf `name` ist dann nicht mehr möglich:

```java
Patient p = new Patient();
p.name = "Anna";    // Compiler-Fehler! name ist private
```

In Python markiert man private Attribute mit `_` oder `__`, aber es ist nur eine Konvention – der Zugriff ist trotzdem möglich. In Java wird der Zugriff **vom Compiler erzwungen**.

## Datenkapselung – Warum?

- **Kontrolle:** Die Klasse kann entscheiden, wie auf ihre Daten zugegriffen wird
- **Validierung:** Ungültige Werte können abgefangen werden
- **Flexibilität:** Die interne Implementierung kann geändert werden, ohne dass externe Code betroffen ist

## Methoden als Schnittstelle

Die Klasse bietet öffentliche Methoden als kontrollierte Schnittstelle:

```java
public class Patient {
    private String name;
    private int alter;

    public void setAlter(int alter) {
        if (alter >= 0 && alter <= 150) {
            this.alter = alter;
        } else {
            System.out.println("Ungültiges Alter!");
        }
    }

    public int getAlter() {
        return alter;
    }
}
```

## Zusammenfassung

### Wichtige Begriffe
- `public` / `private` / `protected`
- Datenkapselung
- Sichtbarkeit
- Package-Private (default)

### Das sollst du können
- Die Zugriffsmodifizierer in Java nennen und anwenden
- Datenkapselung mit `private` umsetzen
- Den Unterschied zur Python-Konvention (`_`, `__`) erklären
