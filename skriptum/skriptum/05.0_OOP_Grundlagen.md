## Klasse als Datentyp

In Java ist eine Klasse immer auch ein **Datentyp**. Du kannst Variable vom Typ einer Klasse deklarieren:

```java

p1 = new Patient("Anna"); // Objekt erzeugen mit new

Patient p2 = new Patient("Ben");  // Deklaration + Initialisierung
```

Der `new`-Operator erzeugt ein neues Objekt. Ohne `new` hast du nur eine Variable, die `null` enthĂ¤lt (vergleichbar mit `None` in Python).

## Attribute und Methoden

**Attribute** (Instanzvariablen) speichern den Zustand eines Objekts, **Methoden** definieren sein Verhalten.

```java

    int alter;

    // Methode

        System.out.println("Name: " + name + ", Alter: " + alter);
    }

```

