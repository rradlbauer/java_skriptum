### UE_09.3_1: Erstes Lambda

Erstelle eine Klasse `ErstesLambda` mit einer `main`-Methode:

1. Definiere ein funktionales Interface `Verdoppler` mit der Methode `int verdopple(int x)`
2. Weise dem Interface ein Lambda zu, das den Parameter verdoppelt
3. Rufe die Methode mit `5` auf und gib das Ergebnis aus (erwartet: 10)

### UE_09.3_2: Built-in funktionale Interfaces

Erstelle eine Klasse `FunktionsInterfaces` und verwende die built-in Interfaces:

1. `Predicate<String>` – prüfe, ob ein String mit "A" beginnt
2. `Consumer<String>` – gib einen String in Großbuchstaben aus
3. `Function<String, Integer>` – ermittle die Länge eines Strings
4. `Supplier<Double>` – liefere eine Zufallszahl (`Math.random()`)
5. Rufe alle vier mit Testdaten auf

### UE_09.3_3: Patienten filtern

Erstelle eine Klasse `PatientFilter` mit einer `Patient`-Klasse (Attribute: `name`, `alter`, `diagnose`):

Gegeben:

```java
List<Patient> patienten = List.of(
    new Patient("Anna", 25, "Grippe"),
    new Patient("Ben", 30, "Diabetes"),
    new Patient("Clara", 22, "Grippe"),
    new Patient("David", 45, "Fraktur")
);
```

1. Verwende `removeIf` mit einem Lambda, um alle Grippe-Patienten zu entfernen
2. Gib die restlichen Patienten aus
3. Erstelle eine `Predicate<Patient>`-Variable für jede Diagnose und verwende sie

### UE_09.3_4: Patienten sortieren

Erstelle eine Klasse `PatientSort` (gleiche `Patient`-Klasse):

1. Verwende `sort` mit Lambda, um Patienten nach Alter zu sortieren
2. Sortiere absteigend nach Alter
3. Sortiere alphabetisch nach Name
4. Sortiere zuerst nach Diagnose, dann nach Alter (verwende `Comparator.comparing(...).thenComparing(...)`)

### UE_09.3_5: Variable Capture

Erstelle eine Klasse `VariableCapture`:

1. Erstelle eine `List<String>` mit 5 Namen
2. Deklariere eine Variable `String titel = "Patient: "` (effectively final)
3. Gib mit `forEach` und Lambda alle Namen mit dem Titel aus
4. Versuche, `titel` nach dem Lambda zu ändern – beobachte den Compiler-Fehler
5. Erkläre in einem Kommentar, was "effectively final" bedeutet

### UE_09.3_6: Lambda vs. anonyme Klasse

Erstelle eine Klasse `VergleichLambda`:

1. Erstelle eine `List<String>` mit 4 Namen
2. Sortiere sie mit einer anonymen `Comparator`-Klasse (wie vor Java 8)
3. Sortiere sie mit einem Lambda
4. Vergleiche die Lesbarkeit beider Varianten in einem Kommentar

### UE_09.3_7: Supplier für Patienten

Erstelle eine Klasse `PatientSupplier`:

1. Verwende `Supplier<Patient>`, um einen Lambda zu schreiben, der einen neuen Patienten erzeugt (zufällige Werte erlaubt)
2. Rufe den Supplier 5-mal auf und gib die erzeugten Patienten aus
3. Verwende einen `Supplier<String>`, um einen Wert aus einer `List<String>` zu liefern
