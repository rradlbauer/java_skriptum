### UE_05.4_1: Patientenverwaltung mit Zähler

Erweitere die Klasse `Patient` um:

- Ein privates statisches Attribut `anzahlPatienten` (int), das bei jedem neuen Patienten erhöht wird
- Eine öffentliche statische Methode `getAnzahlPatienten()`, die den aktuellen Wert zurückgibt
- Eine Instanzmethode `ausgabe()`, die die Patientendaten und die Gesamtzahl der Patienten ausgibt

```java
Patient p1 = new Patient("Anna", 25);
Patient p2 = new Patient("Ben", 30);
Patient p3 = new Patient("Clara", 28);

System.out.println(Patient.getAnzahlPatienten());  // 3
p2.ausgabe();  // "Ben (30) - Gesamt: 3 Patienten"
```

### UE_05.4_2: Medizinische Konstanten

Erstelle eine Klasse `MedizinischeKonstanten` mit `static final`-Konstanten:

- `NORMAL_TEMPERATUR = 37.0`
- `FIEBER_AB = 38.0`
- `UNTERKUEHLUNG_AB = 35.0`
- `MAX_BLUTDRUCK_SYSTOLISCH = 140`
- `MIN_BLUTDRUCK_SYSTOLISCH = 90`
- `BLUTZUCKER_NORMAL_MIN = 70.0`
- `BLUTZUCKER_NORMAL_MAX = 110.0`

Füge eine statische Methode `istFieber(double temperatur)` hinzu, die prüft, ob die Temperatur über dem Fieber-Grenzwert liegt.

### UE_05.4_3: BMI-Rechner als statische Methode

Erstelle eine Klasse `BMIRechner` mit:

- Einer statischen Methode `berechneBMI(double gewichtKg, double groesseM)`, die den BMI berechnet
- Einer statischen Methode `klassifiziereBMI(double bmi)`, die einen String zurückgibt:
  - < 18.5: "Untergewicht"
  - 18.5 - 24.9: "Normalgewicht"
  - 25.0 - 29.9: "Übergewicht"
  - >= 30.0: "Adipositas"
- Verwende die Klasse ohne ein Objekt zu erzeugen (nur über den Klassennamen)

### UE_05.4_4: Statisch vs. Instanz

Analysiere folgenden Code und beantworte die Fragen:

```java
public class Analyse {
    private int instanzWert;
    private static int statischWert;

    public void instanzMethode() {
        System.out.println(instanzWert);     // Frage 1: OK?
        System.out.println(statischWert);    // Frage 2: OK?
    }

    public static void statischMethode() {
        System.out.println(instanzWert);     // Frage 3: OK?
        System.out.println(statischWert);    // Frage 4: OK?
    }
}
```
