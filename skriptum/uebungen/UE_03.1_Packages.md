### UE_03.1_1: Projektstruktur mit Packages

Erstelle folgende Projektstruktur für ein kleines Krankenhaus-System:

```
krankenhaus/
├── src/
│   └── at/
│       └── htl/
│           └── medizin/
│               ├── Patient.java
│               └── Krankenhaus.java
└── bin/
```

Inhalt von `Patient.java`:
```java
package at.htl.medizin;

public class Patient {
    public static void ausgabe() {
        System.out.println("Patient aus dem Package at.htl.medizin");
    }
}
```

Inhalt von `Krankenhaus.java`:
```java
package at.htl.medizin;

public class Krankenhaus {
    public static void main(String[] args) {
        System.out.println("Krankenhaus-System startet...");
        Patient.ausgabe();
    }
}
```

1. Kompiliere beide Dateien mit: `javac -d bin src/at/htl/medizin/*.java`
2. Überprüfe die erzeugte Struktur im `bin`-Ordner
3. Führe das Programm aus: `java -cp bin at.htl.medizin.Krankenhaus`

### UE_03.1_2: Import aus另一个 Package

Erstelle ein zweites Package `at.htl.labor` mit einer Klasse `Labor.java`.
Importiere und verwende die `Labor`-Klasse in `Krankenhaus.java`.

```java
package at.htl.labor;

public class Labor {
    public static void laborDaten() {
        System.out.println("Labordaten werden geladen...");
    }
}
```

Füge in `Krankenhaus.main()` den Aufruf `Labor.laborDaten();` hinzu.
