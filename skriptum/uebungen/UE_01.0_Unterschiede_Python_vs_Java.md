### UE_01.0_1: Multiple Choice

Kreuze die richtigen Antworten an:

1. Was ist der Hauptunterschied zwischen einer interpretierten und einer compilierten Sprache?
   - a) Interpretierte Sprachen sind immer schneller
   - b) Compiler übersetzen Code vor der Ausführung in eine maschinennahe Form
   - c) Es gibt keinen Unterschied

2. Was ist Byte Code?
   - a) Eine besonders kurze Code-Form
   - b) Das Zwischenprodukt nach dem Kompilieren von Java-Code
   - c) Ein Code, der mit Bytes arbeitet

3. Was macht die JVM?
   - a) Sie kompiliert Java-Code
   - b) Sie führt den Byte Code aus und ist für die Plattformunabhängigkeit verantwortlich
   - c) Sie ist ein Texteditor für Java
