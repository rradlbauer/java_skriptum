# UE_09.1 Java Collections – Übungen

[<<](../skriptum/09.1_Collections.md)

### UE_09.1_1: Collection-Methoden anwenden

Erstelle eine Klasse `CollectionDemo` mit einer `main`-Methode:

1. Erstelle eine `Collection<String>` mit einer `ArrayList`
2. Füge die Namen "Anna", "Ben", "Clara", "David", "Anna" hinzu
3. Gib die Größe der Collection aus
4. Prüfe, ob "Clara" enthalten ist
5. Entferne "David"
6. Gib alle Elemente mit einer for-each-Schleife aus
7. Leere die Collection und prüfe, ob sie leer ist

### UE_09.1_2: Iterator manuell verwenden

Erstelle eine Klasse `IteratorDemo`:

1. Erstelle eine `List<String>` mit 5 Patientennamen
2. Verwende einen `Iterator<String>`, um die Liste zu durchlaufen
3. Entferne mit `iterator.remove()` jeden zweiten Namen
4. Gib die restlichen Namen aus

### UE_09.1_3: Autoboxing beobachten

Erstelle eine Klasse `AutoboxingDemo`:

1. Erstelle eine `List<Integer>` und füge 5 int-Werte mit Autoboxing hinzu
2. Lese die Werte aus und weise sie int-Variablen zu (Unboxing)
3. Gib die Summe aller Werte aus

### UE_09.1_4: Collection in Array umwandeln

Erstelle eine Klasse `CollectionToArrayDemo`:

1. Erstelle eine `List<String>` mit Diagnosen
2. Wandle sie mit `toArray()` in ein `Object[]` um
3. Wandle sie mit `toArray(new String[0])` in ein `String[]` um
4. Gib die Arrays aus

### UE_09.1_5: Iterable verstehen

Erstelle eine Klasse `IterableVerstehen`:

1. Erstelle eine `List<String>` mit 3 Elementen
2. Verwende explizit `iterator()` und `hasNext()`/`next()` (statt for-each)
3. Gib den Zustand des Iterators nach jedem Schritt aus
