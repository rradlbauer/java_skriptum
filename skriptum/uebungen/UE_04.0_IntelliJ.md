### UE_04.0_1: Erstes Projekt in IntelliJ

1. Erstelle ein neues IntelliJ-Projekt mit dem Namen `KrankenhausVerwaltung`
2. Wähle dein installiertes JDK aus
3. Erstelle ein Package `at.htl.krankenhaus`
4. Erstelle eine Klasse `Main` mit einer main-Methode
5. Gib folgende Ausgabe aus:

```
=== Krankenhaus-Verwaltung ===
Willkommen im IntelliJ-Projekt!
```

6. Führe das Programm aus (grüner Pfeil)

### UE_04.0_2: IntelliJ-Konfiguration

1. Ändere das Theme (Darcula / IntelliJ Light)

3. Stelle die Schriftgröße auf 16pt ein

Dokumentiere mit Screenshots, was du geändert hast.

### UE_04.0_3: Mit dem Debugger arbeiten

Erstelle eine Klasse `Blutdruck.java` mit folgendem Code:

```java
package at.htl.krankenhaus;

public class Blutdruck {
    public static void main(String[] args) {
        int systolisch = 140;
        int diastolisch = 90;

        System.out.println("Blutdruck: " + systolisch + "/" + diastolisch);

        if (systolisch >= 140 || diastolisch >= 90) {
            System.out.println("Erhöhter Blutdruck!");
        } else {
            System.out.println("Normaler Blutdruck.");
        }

        System.out.println("Messung abgeschlossen.");
    }
}
```

1. Setze einen Breakpoint in Zeile 7 (int systolisch = 140;)
2. Starte den Debugger
3. Gehe Schritt für Schritt durch das Programm (F8)
4. Beobachte, wie sich die Variablen ändern
5. Notiere, welche Ausgaben bei welchen Schritten erscheinen
