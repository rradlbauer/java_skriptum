### UE_11.0_1: Verbindung testen

Schreibe ein Programm `VerbindungsTest.java`, das eine Verbindung zu einer MySQL-Datenbank `medizin` herstellt. Gib bei Erfolg den Datenbanknamen aus, bei Fehler die Fehlermeldung.

Verwende `try-with-resources` und die URL `jdbc:mysql://localhost:3306/medizin`.

### UE_11.0_2: Tabelle anlegen

Erstelle mit einem MySQL-Client (z. B. MySQL Workbench, phpMyAdmin oder der Kommandozeile) folgende Tabelle:

```sql
CREATE TABLE patient (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    alter INT NOT NULL,
    diagnose VARCHAR(200)
);

INSERT INTO patient (name, alter, diagnose) VALUES
('Anna Müller', 34, 'Hypertonie'),
('Ben Bauer', 28, 'Diabetes'),
('Clara Schmidt', 45, 'Fraktur'),
('David Weber', 52, 'Hypertonie'),
('Eva Fischer', 38, 'Asthma');
```

### UE_11.0_3: Alle Patienten abfragen

Schreibe ein Programm `PatientAnzeigen.java`, das alle Patienten aus der Tabelle `patient` abfragt, in `Patient`-Objekte umwandelt und auf der Konsole ausgibt.

### UE_11.0_4: PreparedStatement mit Parameter

Schreibe eine Methode `sucheNachDiagnose(String diagnose)` in einer Klasse `PatientDAO`, die alle Patienten mit einer bestimmten Diagnose zurückgibt. Teste sie mit `"Hypertonie"`.

### UE_11.0_5: INSERT mit generiertem Key

Schreibe eine Methode `einfuegen(Patient p)` in `PatientDAO`, die einen Patienten in die Datenbank einfügt und die generierte ID in das `Patient`-Objekt übernimmt. Teste:

```java
Patient p = new Patient("Greta Schulz", 29, "Migräne");
dao.einfuegen(p);
System.out.println("Gespeichert mit ID: " + p.getId());
```

### UE_11.0_6: UPDATE und DELETE

Ergänze `PatientDAO` um:

1. `void diagnoseAktualisieren(int id, String neueDiagnose)` – aktualisiert die Diagnose eines Patienten
2. `void loeschen(int id)` – löscht einen Patienten anhand seiner ID

Teste beide Methoden.

### UE_11.0_7: Vollständiges CRUD-Menü

Schreibe ein Programm `CrudMenu.java`, das ein Konsolen-Menü anbietet:

```
1 – Alle Patienten anzeigen
2 – Patienten nach Diagnose suchen
3 – Neuen Patienten anlegen
4 – Diagnose aktualisieren
5 – Patienten löschen
0 – Beenden
```

Verwende `Scanner` für die Eingabe und rufe die entsprechenden `PatientDAO`-Methoden auf.
