# UE_09.0 Generics – Übungen

[<<](../skriptum/09.0_Generics.md)

### UE_09.0_1: Box-Klasse mit Generics

Erstelle eine Generic-Klasse `Box<T>` mit:
- Einem Attribut `inhalt` vom Typ `T`
- Getter `getInhalt()` und Setter `setInhalt(T inhalt)`

Erstelle in einer `main`-Methode:
1. Eine `Box<String>` für einen Patientennamen
2. Eine `Box<Integer>` für eine Patientennummer
3. Eine `Box<Patient>` (verwende eine Patient-Klasse mit name, alter)
4. Setze Werte und gib sie aus

### UE_09.0_2: Pair-Klasse mit zwei Typparametern

Erstelle eine Generic-Klasse `Pair<K, V>` mit:
- Attributen `key` (K) und `value` (V)
- Konstruktor, Getter

Verwende sie als `Pair<String, Integer>` für (Name, Alter) und `Pair<String, String>` für (Name, Diagnose).

### UE_09.0_3: Generic-Methoden

Erstelle eine Klasse `GenericMethods` mit folgenden statischen Methoden:

1. `<T> void printArray(T[] array)` – gibt jedes Element eines Arrays aus
2. `<T> T getLetztes(List<T> list)` – gibt das letzte Element einer Liste zurück
3. Teste beide Methoden mit `String[]`, `Integer[]`, `List<String>`, `List<Double>`

### UE_09.0_4: Bounded Type Parameters

Erstelle eine Klasse `BoundedGenerics`:

1. Schreibe eine Methode `<T extends Comparable<T>> T max(T a, T b)`, die das größere von zwei Elementen zurückgibt
2. Teste mit `Integer`, `String`, `Double`
3. Versuche, die Methode mit einer eigenen Klasse `Patient` aufzurufen, die `Comparable` nicht implementiert – beobachte den Compiler-Fehler
4. Lass `Patient` dann `Comparable<Patient>` implementieren (vergleiche nach Alter)

### UE_09.0_5: Wildcards verstehen

Erstelle eine Klasse `WildcardDemo`:

1. Schreibe eine Methode `void printAll(List<?> list)`, die alle Elemente ausgibt
2. Schreibe eine Methode `double sumNumbers(List<? extends Number> list)`, die die Summe berechnet
3. Schreibe eine Methode `void addInts(List<? super Integer> list)`, die die Zahlen 1-5 hinzufügt
4. Teste alle Methoden mit `List<Integer>`, `List<Double>`, `List<String>`, `List<Object>`
5. Erkläre in Kommentaren, warum manche Aufrufe nicht funktionieren

### UE_09.0_6: Diamond-Operator

Erstelle eine Klasse `DiamondDemo`:

1. Erstelle eine `List<String>` mit `new ArrayList<>()` (Diamond-Operator)
2. Erstelle eine `Map<String, List<Integer>>` mit `new HashMap<>()`
3. Füge Werte hinzu und gib sie aus
4. Schreibe dazu einen Kommentar, wie der Code vor Java 7 ausgesehen hätte
