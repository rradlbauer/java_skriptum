### UE_05.3_1: Dosierungsrechner

Erstelle eine Klasse `Dosierungsrechner` mit überladenen Methoden `berechneDosis()`:

1. `berechneDosis(double gewicht, double mgProKg)` – einfache Dosierung
2. `berechneDosis(double gewicht, double mgProKg, int alter)` – für Kinder unter 12 Jahren wird die Dosis halbiert
3. `berechneDosis(String wirkstoff, double gewicht)` – schlägt anhand des Wirkstoffs den mgProKg-Faktor nach (verwende if/else)
   - "Ibuprofen" → 10 mg/kg
   - "Paracetamol" → 15 mg/kg
   - "Amoxicillin" → 20 mg/kg
   - Default → 5 mg/kg

Teste alle drei Varianten in der `main`-Methode.

### UE_05.3_2: Überladene Konstruktoren

Erstelle eine Klasse `Arzttermin` mit folgenden Konstruktoren:

1. `Arzttermin(String patient, String datum)` – Standardtermin (30 Minuten)
2. `Arzttermin(String patient, String datum, int dauer)` – Termin mit benutzerdefinierter Dauer
3. `Arzttermin(String patient, String datum, int dauer, String fachrichtung)` – vollständiger Termin

Verwende `this(...)`, um Wiederholungen zu vermeiden.
Füge eine Methode `ausgabe()` hinzu, die alle Termindetails anzeigt.

### UE_05.3_3: Maximalen Puls berechnen

Die maximale Herzfrequenz kann auf verschiedene Arten berechnet werden:

1. Einfache Formel: `220 - alter` (für Erwachsene)
2. Genauere Formel: `208 - 0.7 * alter` (Tanaka-Formel)
3. Für Sportler: `205 - 0.5 * alter`

Erstelle überladene Methoden `maxPuls()`:
- `maxPuls(int alter)` – verwendet die einfache Formel
- `maxPuls(int alter, boolean genau)` – wenn true, verwende Tanaka-Formel
- `maxPuls(int alter, boolean genau, boolean sportler)` – wenn sportler true, verwende Sportler-Formel
