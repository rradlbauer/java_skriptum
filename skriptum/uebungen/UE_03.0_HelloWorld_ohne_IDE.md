### UE_03.0_1: Erste Schritte mit Kommandozeile

1. Erstelle mit einem Texteditor (Notepad, VS Code, ...) eine Datei `HalloMedizin.java` mit folgendem Inhalt:

```java
public class HalloMedizin {
    public static void main(String[] args) {
        System.out.println("Willkommen in der medizinischen Informatik!");
    }
}
```

2. Kompiliere die Datei mit `javac HalloMedizin.java`
3. Überprüfe, ob eine `HalloMedizin.class`-Datei entstanden ist
4. Führe das Programm mit `java HalloMedizin` aus
5. Dokumentiere jeden Schritt und die Ausgaben

### UE_03.0_2: Patientendaten ausgeben

Schreibe ein Programm `PatientAusgabe.java`, das folgende Patientendaten auf der Konsole ausgibt:

```
=== Patientendaten ===
Name: Anna Müller
Alter: 34
Diagnose: Hypertonie
Aufnahmedatum: 15.01.2024
```

Kompiliere und führe das Programm aus.

### UE_03.0_3: Scanner – Patientendaten einlesen

Schreibe ein Programm `PatientEingabe.java`, das:

1. Den Benutzer mit `Scanner` nach Name, Alter und Diagnose fragt
2. Die eingegebenen Daten formatiert mit `System.out.printf()` ausgibt
3. Den Scanner nach der Verwendung schließt

Beispiel-Ablauf:

```
Bitte Namen eingeben: Anna Müller
Bitte Alter eingeben: 34
Bitte Diagnose eingeben: Hypertonie

=== Patientendaten ===
Name:   Anna Müller
Alter:  34 Jahre
Diagnose: Hypertonie
```

### UE_03.0_4: Fehler erkennen

Gegeben ist folgender Code:

```java
public class FehlerTest {
    public static void main(String[] args) {
        System.out.println("Start der Diagnose")
        int patientenId = "MK-0815"
    }
}
```

1. Versuche, diesen Code zu kompilieren
2. Notiere die Fehlermeldung(en)
3. Korrigiere die Fehler und führe das Programm aus
