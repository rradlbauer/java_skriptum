### UE_05.0_1: Patient-Klasse

Erstelle eine Klasse `Patient` im Package `at.htl.medizin` mit folgenden Attributen:

- `diagnose` (String)

Füge eine Methode `ausgabe()` hinzu, die alle Patientendaten formatiert ausgibt.

### UE_05.0_3: Von Python zu Java

Gegeben ist folgender Python-Code:

```python
class Patient:
    def __init__(self, name, alter):
        self.name = name
        self.alter = alter

    def bmi(self, gewicht, groesse):
        return gewicht / (groesse ** 2)

p = Patient("Anna", 25)
print(p.bmi(70, 1.75))
```

Übersetze diesen Code nach Java. Achte auf:
- Die Klasse heißt genauso wie die Datei
- Konstruktor hat keinen Rückgabetyp
- `self` wird zu `this`
- `**` gibt es nicht – verwende `Math.pow()` oder einfache Multiplikation
