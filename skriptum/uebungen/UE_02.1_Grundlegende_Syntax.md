### UE_02.1_1: Syntaxfehler finden

Der folgende Code enthält mehrere Fehler. Finde und korrigiere sie:

```java
public class KorperTemperatur {
    public static void main(String[] args) {
        double patientenTemperatur = 38.5
        int herzfrequenz = 72

        if patientenTemperatur > 37.5 {
            System.out.println("Patient hat Fieber")
        }
        System.out.println("Herzfrequenz: " + herzfrequenz)
    }
}
```

Schreibe die korrigierte Version in eine Datei `KoerperTemperatur.java`, kompiliere und führe sie aus.

### UE_02.1_2: Variablen deklarieren

Deklariere und initialisiere Variablen für folgende Patientendaten mit passenden Typen:

- Patienten-ID (ganze Zahl, 5-stellig)
- Körpertemperatur (Gleitkommazahl)
- Hat Allergien (ja/nein)
- Blutgruppe (einzelnes Zeichen wie 'A', 'B', '0')
- Versicherungsnummer (sehr große Zahl)

### UE_02.1_3: Konstanten in der Medizin

Definiere Konstanten mit `final` für folgende medizinische Grenzwerte:

- Normale Körpertemperatur: 37.0 °C
- Fieber ab: 38.0 °C
- Maximaler Blutdruck (systolisch): 140 mmHg
- Minimaler Blutdruck (systolisch): 90 mmHg

### UE_02.1_4: Blockstruktur

Schreibe ein Programm mit einer if-else-Anweisung, die anhand der Körpertemperatur eine der folgenden Meldungen ausgibt:
- Unter 36.0: "Untertemperatur"
- 36.0 bis 37.5: "Normaltemperatur"
- Über 37.5: "Fieber"
