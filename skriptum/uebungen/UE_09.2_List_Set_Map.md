# UE_09.2 List, Set, Map – Übungen

[<<](../skriptum/09.2_List_Set_Map.md)

### UE_09.2_1: Patientenverwaltung mit List

Erstelle eine Klasse `PatientenListe`:

1. Erstelle eine `List<Patient>` (`ArrayList`) für eine Patientenliste
2. Füge 5 Patienten hinzu
3. Gib den dritten Patienten aus (Index 2)
4. Ändere den Namen des zweiten Patienten
5. Entferne den letzten Patienten
6. Gib alle Patienten mit einer for-each-Schleife aus

### UE_09.2_2: Duplikate entfernen mit Set

Erstelle eine Klasse `DiagnosenSet`:

1. Erstelle eine `List<String>` mit Diagnosen (mehrere Duplikate)
2. Überführe die Liste in ein `HashSet<String>` – beobachte, dass Duplikate verschwinden
3. Gib das Set aus
4. Überführe die Liste in ein `TreeSet<String>` – beobachte die Sortierung

### UE_09.2_3: Patienten-Verzeichnis mit Map

Erstelle eine Klasse `PatientenVerzeichnis`:

1. Erstelle eine `Map<String, Patient>` (`HashMap`) – Schlüssel ist die SV-Nummer
2. Füge 3-4 Patienten mit ihren SV-Nummern hinzu
3. Rufe einen Patienten über seine SV-Nummer ab
4. Prüfe, ob eine bestimmte SV-Nummer vorhanden ist
5. Gib alle Einträge mit `entrySet()` aus
6. Gib alle Namen mit `keySet()` aus
7. Gib alle Alter mit `values()` aus

### UE_09.2_4: equals() und hashCode()

Erstelle eine Klasse `PatientMitSvnr` mit Attributen `svnr` (String) und `name` (String):

1. Überschreibe `equals()` so, dass zwei Patienten gleich sind, wenn ihre SV-Nummer übereinstimmt
2. Überschreibe `hashCode()` entsprechend
3. Erstelle zwei `PatientMitSvnr`-Objekte mit derselben SV-Nummer
4. Füge beide in ein `HashSet` ein – es sollte nur eines enthalten sein
5. Kommentiere `equals()` und `hashCode()` aus und beobachte den Unterschied

### UE_09.2_5: TreeMap für sortierte Ausgabe

Erstelle eine Klasse `SortierteMapDemo`:

1. Erstelle eine `TreeMap<String, Integer>` mit 5 Namen und Altern
2. Gib alle Einträge in einer Schleife aus – sie sollten alphabetisch sortiert sein
3. Erstelle eine `HashMap<String, Integer>` mit denselben Daten
4. Gib auch diese aus – die Reihenfolge ist anders (nicht vorhersagbar)
5. Vergleiche die Ausgaben
