### UE_07.1_1: Abstrakte Klasse Untersuchung

Erstelle eine abstrakte Klasse `Untersuchung`:

```java
public abstract class Untersuchung {
    protected String patientName;
    protected String datum;

    public Untersuchung(String patientName, String datum) {
        this.patientName = patientName;
        this.datum = datum;
    }

    public abstract void durchfuehren();
    public abstract String getErgebnis();

    public String getPatientName() {
        return patientName;
    }
}
```

Erstelle zwei konkrete Unterklassen:

1. `Blutuntersuchung`:
   - Zusätzliche Attribute: `blutwert` (String), `messwert` (double)
   - `durchfuehren()` gibt aus: "Blutuntersuchung für [patientName]: [blutwert] = [messwert]"
   - `getErgebnis()` gibt einen String mit dem Ergebnis zurück

2. `EKG`:
   - Zusätzliche Attribute: `herzfrequenz` (int), `rhythmus` (String, z. B. "Sinusrhythmus")
   - `durchfuehren()` gibt aus: "EKG für [patientName]: [frequenz] bpm, [rhythmus]"
   - `getErgebnis()` gibt einen String zurück

### UE_07.1_2: Interface Behandelbar

Erstelle ein Interface `Behandelbar`:

```java
public interface Behandelbar {
    void behandeln();
    String getDiagnose();
}
```

Implementiere das Interface in zwei unterschiedlichen Klassen:

1. `Patient` (hat Attribute: `name`, `diagnose`, `alter`)
   - `behandeln()` gibt aus: "Patient [name] wird behandelt für: [diagnose]"
   
2. `Tier` (hat Attribute: `tierart`, `name`, `diagnose`)
   - `behandeln()` gibt aus: "[tierart] [name] wird behandelt für: [diagnose]"

Zeige in einer `main`-Methode, dass beide Klassen über das Interface angesprochen werden können.

### UE_07.1_3: Impfbar und Untersuchbar

Erstelle zwei Interfaces:

```java
public interface Impfbar {
    void impfen(String impfstoff);
    boolean istGeimpft();
}

public interface Untersuchbar {
    void untersuchen(String art);
    String letzterBefund();
}
```

Erstelle eine Klasse `Hund`, die beide Interfaces implementiert (zusätzliche Attribute: `name`, `geimpft`, `letzteUntersuchung`).

Erstelle in der `main`-Methode einen Hund und rufe Methoden beider Interfaces auf.
