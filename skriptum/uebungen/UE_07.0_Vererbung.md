### UE_07.0_1: Personal-Hierarchie im Krankenhaus

Erstelle folgende Vererbungshierarchie:

**Oberklasse: `Personal`**
- Attribute: `name` (String), `personalnummer` (String), `gehalt` (double)
- Konstruktor mit allen Attributen
- Methode `arbeite()` gibt aus: `"[name] arbeitet im Krankenhaus."`

**Unterklasse: `Arzt` extends `Personal`**
- Zusätzliches Attribut: `fachrichtung` (String)
- Konstruktor mit allen Attributen (verwende `super()`)
- Überschreibe `arbeite()`: `"[name] behandelt Patienten in der Fachrichtung [fachrichtung]."`

**Unterklasse: `Krankenpfleger` extends `Personal`**
- Zusätzliches Attribut: `station` (String)
- Konstruktor mit allen Attributen
- Überschreibe `arbeite()`: `"[name] pflegt Patienten auf der Station [station]."`

**Test-Klasse: `KrankenhausTest`**
- Erstelle je einen Arzt und einen Krankenpfleger
- Rufe `arbeite()` auf beiden Objekten auf
- Weise beide einer `Personal`-Variable zu und rufe erneut `arbeite()` auf

### UE_07.0_2: Medikamenten-Verwaltung

**Oberklasse: `Medikament`**
- Attribute: `name`, `wirkstoff`, `preis`
- Konstruktor, Getter
- Methode `beschreibung()`: gibt "Medikament: [name] ([wirkstoff])" zurück

**Unterklasse: `VerschreibungspflichtigesMedikament` extends `Medikament`**
- Zusätzliches Attribut: `rezeptPflichtig` (immer true)
- Überschreibe `beschreibung()`: hängt " - Rezeptpflichtig!" an

**Unterklasse: `FreiverkaeuflichesMedikament` extends `Medikament`**
- Zusätzliches Attribut: `empfohleneTagesdosis` (String)
- Überschreibe `beschreibung()`: hängt " - Frei verkäuflich. Empfohlene Dosis: [dosis]" an

### UE_07.0_3: Polymorphie in der Praxis

Erstelle ein Array vom Typ `Personal`, das sowohl Ärzte als auch Pfleger enthält:

```java
Personal[] team = new Personal[3];
team[0] = new Arzt("Dr. Meier", "A-001", 65000.0, "Kardiologie");
team[1] = new Krankenpfleger("Schmidt", "P-002", 38000.0, "Intensiv");
team[2] = new Arzt("Dr. Bauer", "A-002", 72000.0, "Neurologie");
```

Iteriere über das Array und rufe für jedes Element `arbeite()` auf. Dokumentiere die Ausgabe und erkläre, warum die richtige Methode aufgerufen wird.
